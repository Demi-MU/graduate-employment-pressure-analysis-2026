# 指标体系：硕士毕业生就业压力的供需错配分析

## 一、指标体系设计目的

本项目的核心目标，是解释2026年前后部分硕士毕业生为何会产生“读完研反而更难找工作”的求职体感。为了避免将这一现象停留在个体经验或情绪表达层面，项目需要建立一套相对清晰、可复现、可解释的指标体系，用于刻画高校毕业生供给、招聘岗位需求、高学历岗位吸纳能力、学历错配程度以及读研机会成本。

本指标体系并不试图用单一指标解释全部就业困难。高校毕业生就业问题同时受到宏观经济、产业周期、教育扩张、岗位结构、专业设置、城市分布和个体预期等多重因素影响。因此，本项目采用多维指标组合的方式，将“就业难”拆解为四个层面：供给压力、需求吸纳、结构错配和机会成本。通过这些指标，可以更系统地观察硕士毕业生就业压力的形成机制。

## 二、指标设计原则

本项目的指标设计遵循三个基本原则。

首先，指标必须服务于研究问题。本文关注的不是一般意义上的就业率，而是硕士毕业生在应届就业市场中所面临的结构性压力。因此，指标设计应重点刻画研究生毕业供给是否增加、硕士岗位是否同步扩张、不同产业和城市对硕士学历的吸纳能力是否存在分化，以及读研三年是否真正转化为职业资本。

其次，指标必须具有可解释性。由于本项目最终需要转化为公开报告和讲解视频，指标不能过度复杂。每个指标都应能够对应一个清晰的问题。例如，“硕士岗位吸纳比”回答的是市场中有多少应届岗位真正要求硕士学历；“学历错配指数”回答的是硕士毕业生供给占比与硕士岗位需求占比之间是否存在偏离；“读研机会成本”回答的是读研路径与本科直接就业路径之间的相对收益差异。

最后，指标必须保留口径边界。招聘数据、就业报告和官方统计数据之间存在口径差异，不同平台的岗位样本也不能完全代表全国就业市场。因此，所有指标都应明确数据来源、统计范围、时间窗口和适用边界。对于无法获得完整总体数据的指标，本项目将采用公开报告和岗位样本进行近似估计，并在报告中说明其局限性。

## 三、供给侧指标

供给侧指标用于刻画高校毕业生进入就业市场的规模变化。该类指标回答的问题是：2020年至2026年期间，高校毕业生和研究生毕业生是否显著增加？硕士毕业生在全部毕业生中的相对占比是否发生变化？

### 3.1 高校毕业生规模

高校毕业生规模是衡量应届就业市场供给压力的基础指标。该指标表示某一年或某一届进入就业市场的普通高校毕业生总人数。

计算方式为：

高校毕业生规模 = 当年普通高校毕业生总人数

该指标主要用于展示2020年至2026年高校毕业生总量变化。需要注意的是，“年度毕业人数”和“某届毕业生预计人数”可能存在口径差异。教育部在就业季前公布的通常是某一届毕业生预计规模，而教育统计公报中公布的可能是年度实际毕业人数。本项目在使用该指标时，应明确采用的是“预计届别口径”还是“年度统计口径”。

### 3.2 研究生毕业生规模

研究生毕业生规模用于衡量高学历毕业生供给变化。该指标表示某一年研究生教育层次毕业生的总人数，通常包括硕士和博士毕业生。

计算方式为：

研究生毕业生规模 = 当年硕士毕业生人数 + 当年博士毕业生人数

在数据条件允许的情况下，本项目应进一步拆分硕士毕业生和博士毕业生。由于本项目重点关注硕士就业压力，因此硕士毕业生规模比研究生总体规模更具解释力。

### 3.3 硕士毕业生规模

硕士毕业生规模是本项目供给侧分析中最重要的学历分层指标。该指标表示某一年或某一届硕士研究生毕业人数。

计算方式为：

硕士毕业生规模 = 当年硕士研究生毕业人数

该指标用于观察硕士毕业生供给是否快速增加。如果硕士毕业生规模增长较快，而招聘市场中要求硕士学历的岗位没有同步增长，则可能导致硕士学历竞争加剧。

### 3.4 毕业生同比增速

毕业生同比增速用于衡量某类毕业生规模相对于上一年的增长幅度。该指标可以分别用于高校毕业生总人数、研究生毕业生人数和硕士毕业生人数。

计算方式为：

毕业生同比增速 = 当年毕业生人数 / 上一年毕业生人数 - 1

该指标适合用于展示就业市场供给压力的变化速度。相比绝对规模，同比增速能够更清楚地反映某一学历层次毕业生是否处于快速扩张阶段。

### 3.5 研究生毕业生占比

研究生毕业生占比用于衡量高学历毕业生在高校毕业生总体中的相对比重。

计算方式为：

研究生毕业生占比 = 研究生毕业生人数 / 高校毕业生总人数

如果该比例持续上升，说明高校毕业生结构正在向更高学历层次移动。这一变化本身并不等同于学历贬值，但它意味着研究生学历的相对稀缺性下降，企业在招聘时可能会进一步强化对学校、专业、经验和技能的筛选。

### 3.6 硕士毕业生占比

硕士毕业生占比用于衡量硕士毕业生在高校毕业生总体中的相对规模。

计算方式为：

硕士毕业生占比 = 硕士毕业生人数 / 高校毕业生总人数

该指标是后续计算硕士学历错配指数的重要基础。它反映的是供给侧中硕士毕业生的比例，而不是市场对硕士学历的真实需求。

## 四、需求侧指标

需求侧指标用于刻画招聘市场对应届毕业生的岗位吸纳能力。该类指标回答的问题是：市场中有多少岗位面向应届生？这些岗位要求什么学历？其中有多少岗位真正需要硕士？

### 4.1 应届岗位数

应届岗位数表示在某一时间窗口内，明确面向应届毕业生或接受应届毕业生的岗位数量。

计算方式为：

应届岗位数 = 满足应届岗位判定规则的岗位总数

在实际操作中，应届岗位可以分为强应届岗位和弱应届岗位。强应届岗位通常包含“应届”“校招”“校园招聘”“2026届”“毕业生”“管培生”等关键词。弱应届岗位则包括“经验不限”“1年以内”“可接受应届生”等表述。

为了保证指标稳健性，本项目可以分别计算两个版本：

强应届岗位数 = 明确包含应届或校招相关关键词的岗位数

广义应届岗位数 = 强应届岗位数 + 经验不限且接受低年限候选人的岗位数

在正式报告中，应明确说明采用的是狭义口径还是广义口径。

### 4.2 本科及以上岗位数

本科及以上岗位数表示招聘文本中明确要求本科及以上学历的应届岗位数量。

计算方式为：

本科及以上岗位数 = 学历要求为本科、本科及以上、硕士、硕士及以上、博士的应届岗位数

该指标用于观察本科作为应届岗位主流学历门槛的程度。很多企业招聘中写“本科及以上”，并不意味着该岗位真正需要硕士，而是表示本科已经满足基本要求。因此，本科及以上岗位不能被视为硕士岗位。

### 4.3 硕士及以上岗位数

硕士及以上岗位数表示招聘文本中明确要求硕士或博士学历的应届岗位数量。

计算方式为：

硕士及以上岗位数 = 学历要求为硕士、硕士及以上或博士的应届岗位数

该指标是计算硕士硬性吸纳比的分子。需要注意的是，“硕士优先”“研究生优先”不应直接归入硬性要求，而应单独标记为偏好要求。

### 4.4 硕士优先岗位数

硕士优先岗位数表示招聘文本中未强制要求硕士学历，但明确出现“硕士优先”“研究生优先”“硕博优先”等表述的岗位数量。

计算方式为：

硕士优先岗位数 = 学历要求未强制限定硕士及以上，但岗位描述中出现硕士偏好表述的应届岗位数

该指标用于捕捉企业对硕士学历的软性偏好。它与硕士及以上岗位数共同构成硕士偏好吸纳比的分子。

### 4.5 经验不限岗位数

经验不限岗位数表示招聘文本中没有明确要求工作经验，或明确写明“经验不限”的岗位数量。

计算方式为：

经验不限岗位数 = 经验要求字段为经验不限、无经验要求或可接受应届生的岗位数

该指标用于识别应届毕业生可能进入的岗位池。但需要注意，经验不限岗位并不一定等于应届岗位。一些企业虽然写经验不限，但实际招聘时可能仍然偏好有实习或项目经验的候选人。

### 4.6 高薪岗位数

高薪岗位数用于衡量应届岗位中的高质量岗位供给。由于不同行业和城市薪资差异较大，本项目不建议使用全国统一标准定义高薪岗位，而应采用相对口径。

可以采用两种计算方式：

高薪岗位数 = 薪资中位数高于样本整体75分位数的岗位数

或：

高薪岗位数 = 薪资中位数高于同城市同岗位类型75分位数的岗位数

第二种方式更加严谨，因为它控制了城市和岗位类型差异。对于视频传播场景，也可以采用更直观的薪资阈值，但必须说明该阈值只是情景设定。

## 五、高学历岗位吸纳指标

高学历岗位吸纳指标是本项目的核心指标组，用于衡量招聘市场对硕士学历的真实需求强度。

### 5.1 硕士硬性吸纳比

硕士硬性吸纳比表示在应届岗位中，明确要求硕士及以上学历的岗位占比。

计算方式为：

硕士硬性吸纳比 = 明确要求硕士及以上学历的应届岗位数 / 应届岗位总数

该指标回答的问题是：在应届招聘市场中，有多少岗位必须由硕士及以上学历候选人来竞争？

如果该指标较低，说明多数应届岗位并不强制要求硕士学历。即使硕士毕业生数量增加，也不意味着市场会提供同等规模的硕士专属岗位。该指标是解释“高学历供给增加但就业体感未改善”的关键证据。

### 5.2 硕士偏好吸纳比

硕士偏好吸纳比表示在应届岗位中，明确要求硕士及以上学历，或虽然不强制要求但写明硕士优先的岗位占比。

计算方式为：

硕士偏好吸纳比 = 明确要求硕士及以上学历或写明硕士优先的应届岗位数 / 应届岗位总数

该指标比硕士硬性吸纳比更宽松，能够反映企业对硕士学历的软性偏好。比较硕士硬性吸纳比和硕士偏好吸纳比，可以观察企业到底是“必须要硕士”，还是“有硕士更好”。

如果两者差距较大，说明硕士学历在部分岗位中主要作为加分项，而不是准入门槛。对于求职者而言，这意味着硕士学历可能提高竞争概率，但不能单独保证岗位匹配。

### 5.3 行业硕士吸纳比

行业硕士吸纳比表示某一行业中，明确要求硕士及以上学历的应届岗位占该行业全部应届岗位的比例。

计算方式为：

行业硕士吸纳比 = 某行业要求硕士及以上学历的应届岗位数 / 某行业应届岗位总数

该指标用于比较不同行业对硕士学历的需求差异。通常而言，教育科研、医药研发、半导体、部分金融研究、公共部门和高端制造研发岗位可能具有更高的硕士吸纳比，而销售、运营、客服、普通职能和一般服务类岗位的硕士吸纳比可能较低。

该指标有助于说明：硕士就业压力并不是均匀发生的，而是高度依赖行业结构。

### 5.4 城市硕士吸纳比

城市硕士吸纳比表示某一城市中，明确要求硕士及以上学历的应届岗位占该城市全部应届岗位的比例。

计算方式为：

城市硕士吸纳比 = 某城市要求硕士及以上学历的应届岗位数 / 某城市应届岗位总数

该指标用于比较不同城市对硕士学历的岗位吸纳能力。一线城市和部分产业型新一线城市可能提供更多高学历岗位，但同时也吸引更多硕士毕业生集中竞争。因此，城市硕士吸纳比不能单独解释就业难度，还需要结合岗位数量、毕业生流入倾向和竞争强度共同分析。

### 5.5 行业—城市硕士吸纳比

行业—城市硕士吸纳比表示某一城市某一行业中，明确要求硕士及以上学历的应届岗位占该城市该行业全部应届岗位的比例。

计算方式为：

行业—城市硕士吸纳比 = 某城市某行业要求硕士及以上学历的应届岗位数 / 某城市某行业应届岗位总数

这是本项目最适合做热力图的指标。它可以展示不同城市、不同行业对硕士学历的吸纳能力差异。例如，北京的教育科研、上海的金融、深圳的半导体、杭州的互联网、成都的电子信息等行业城市组合，可能呈现完全不同的硕士需求结构。

该指标能够帮助视频观众直观看到：问题并不是“硕士有没有用”，而是“硕士在哪些行业和城市更容易被市场定价”。

## 六、学历错配指标

学历错配指标用于比较毕业生供给结构与岗位需求结构之间的偏离。该类指标回答的问题是：硕士毕业生在供给端的占比，是否高于招聘市场中硕士岗位的占比？

### 6.1 硕士学历错配指数

硕士学历错配指数是本项目解释高学历就业压力的重要指标。

计算方式为：

硕士学历错配指数 = 硕士毕业生供给占比 - 硕士岗位需求占比

其中：

硕士毕业生供给占比 = 硕士毕业生人数 / 高校毕业生总人数

硕士岗位需求占比 = 要求硕士及以上学历的应届岗位数 / 应届岗位总数

如果该指数为正，说明硕士毕业生在供给端的占比高于硕士岗位在需求端的占比，可能存在硕士供给相对过剩或岗位吸纳不足的问题。

如果该指数持续扩大，说明硕士毕业生供给增长可能快于硕士岗位需求增长，硕士学历的市场稀缺性可能下降。

如果该指数为负，则说明招聘市场中硕士岗位占比高于硕士毕业生供给占比，可能意味着某些行业或城市对硕士学历存在较强需求。

需要注意的是，该指标是结构性指标，不应被简单解读为“硕士过剩”。它只能说明某一口径下供给占比与需求占比之间存在差异，不能直接代表每一个专业或每一个个体的就业结果。

### 6.2 本科岗位主导指数

本科岗位主导指数用于衡量应届岗位市场中本科及以上学历作为主流门槛的程度。

计算方式为：

本科岗位主导指数 = 本科及以上岗位数 / 应届岗位总数

如果该指标较高，而硕士硬性吸纳比较低，说明大多数应届岗位的基本学历门槛仍然停留在本科层次。此时，硕士毕业生虽然可以投递这些岗位，但并不会因为学历更高而自动获得明显优势。

### 6.3 学历下沉竞争指数

学历下沉竞争指数用于衡量硕士毕业生向本科岗位竞争的可能程度。

计算方式可设定为：

学历下沉竞争指数 = 本科及以上岗位数 / 硕士及以上岗位数

该指标越高，说明市场中本科及以上岗位远多于硕士及以上岗位，硕士毕业生可能更频繁地进入本科岗位池竞争。该指标不是正式统计指标，而是项目内部构造的解释性指标，用于辅助说明“高学历向下竞争”的现象。

## 七、结构性错配指标

结构性错配指标用于进一步刻画就业困难的具体来源。高校毕业生就业压力并不只来自学历层次，也来自专业、行业、城市和预期之间的不匹配。

### 7.1 行业岗位占比

行业岗位占比表示某一行业的应届岗位数占全部应届岗位数的比例。

计算方式为：

行业岗位占比 = 某行业应届岗位数 / 全部应届岗位数

该指标用于观察应届岗位主要集中在哪些行业。如果毕业生偏好的行业岗位占比下降，而其他行业岗位占比上升，就可能出现“岗位存在但不符合预期”的现象。

### 7.2 城市岗位占比

城市岗位占比表示某一城市的应届岗位数占全部样本应届岗位数的比例。

计算方式为：

城市岗位占比 = 某城市应届岗位数 / 全部应届岗位数

该指标用于观察岗位在空间上的集中程度。高校毕业生往往集中流向一线城市和新一线城市，但这些城市的岗位竞争也更激烈。因此，城市岗位占比需要与城市投递集中度、毕业生流入意愿和薪资水平结合分析。

### 7.3 专业岗位匹配度

专业岗位匹配度用于衡量某类专业毕业生与相关岗位之间的匹配程度。由于本项目未必能够获得完整专业毕业生数据和岗位专业要求数据，因此该指标可作为扩展指标。

一种可行的计算方式为：

专业岗位匹配度 = 某专业相关岗位数 / 某专业毕业生人数

如果某专业毕业生人数较多，而相关岗位数较少，则可能存在专业供给过剩或岗位不足的问题。

在实际操作中，专业岗位匹配度需要建立专业—岗位映射表。例如，计算机类专业可对应软件开发、算法、数据、测试、产品技术等岗位；金融类专业可对应投研、风控、银行、证券、保险、金融科技等岗位；教育类专业可对应教师、教研、培训、辅导员、教育管理等岗位。

### 7.4 岗位经验门槛指数

岗位经验门槛指数用于衡量应届生进入某类岗位的难度。

计算方式为：

岗位经验门槛指数 = 要求1年以上工作经验的岗位数 / 全部岗位数

对于应届生而言，经验要求是影响岗位可进入性的重要因素。如果一个行业岗位数量很多，但大部分岗位要求工作经验，则它对真正应届毕业生的吸纳能力有限。

### 7.5 岗位质量指数

岗位质量指数用于综合衡量岗位吸引力。由于岗位质量具有多维属性，本项目可以采用简化方式构建综合指标。

可纳入的维度包括：

薪资水平、是否专业相关、是否接受应届生、是否明确培养机制、是否有稳定雇佣关系、企业类型、岗位职能、城市层级。

一种简化计算方式为：

岗位质量指数 = 标准化薪资得分 + 专业匹配得分 + 应届友好得分 + 稳定性得分

该指标适合作为扩展分析，不建议作为项目早期核心指标。原因是岗位质量涉及较多主观赋权，若处理不当容易影响结论可信度。

## 八、队列比较指标

队列比较指标用于解释“本科毕业时”和“研究生毕业时”之间的真实差异。该类指标关注的不是静态学历比较，而是不同时间路径下个体职业资本的变化。

### 8.1 本科直接就业路径

本科直接就业路径表示学生在本科毕业后直接进入劳动力市场，并在三年内积累工作经验和收入。

核心变量包括：

初始月薪、薪资增长率、三年累计收入、工作经验年限、行业经验、项目经历、晋升机会。

该路径用于模拟“如果本科毕业后直接工作，到2026年可能处于什么竞争位置”。

### 8.2 读研后就业路径

读研后就业路径表示学生本科毕业后进入研究生阶段，并在三年后以硕士应届生身份重新进入就业市场。

核心变量包括：

研究生学费、生活成本、奖助学金、实习收入、硕士毕业起薪、学校平台、专业壁垒、科研成果、实习经历、校招资格。

该路径用于模拟“读研三年是否转化为更强职业资本”。

### 8.3 职业资本增量

职业资本增量用于描述读研期间形成的就业竞争力提升。

可以用定性方式表示为：

职业资本增量 = 学历提升 + 学校平台提升 + 专业壁垒提升 + 项目经历提升 + 实习经历提升 + 行业资源提升

如果读研期间主要增加的是学历标签，而没有明显增加项目能力、实习经历或专业壁垒，则职业资本增量可能较低。此时，硕士毕业生在就业市场上可能难以相对于本科毕业后直接工作的人形成优势。

## 九、机会成本指标

机会成本指标用于解释读研决策的经济与职业代价。它回答的问题是：读研三年后获得的就业收益，是否足以覆盖三年直接工作的机会成本？

### 9.1 本科直接工作三年累计收入

计算方式为：

本科直接工作三年累计收入 = 第一年收入 + 第二年收入 + 第三年收入

如果假设本科毕业初始月薪为 S，年薪增长率为 g，则可表示为：

本科直接工作三年累计收入 = 12S + 12S(1+g) + 12S(1+g)^2

该指标用于估计本科毕业后直接进入市场可能获得的显性收入。

### 9.2 读研期间净成本

读研期间净成本表示研究生阶段的经济支出扣除奖助学金和实习收入后的成本。

计算方式为：

读研期间净成本 = 学费 + 生活成本 - 奖助学金 - 实习收入

该指标不包括心理成本、时间成本和不确定性成本，但可以作为显性经济成本估计。

### 9.3 硕士毕业薪资增量

硕士毕业薪资增量用于衡量硕士学历相对于本科直接就业路径的薪资提升。

计算方式为：

硕士毕业薪资增量 = 硕士毕业起薪 - 本科直接就业三年后的当前薪资

如果硕士毕业起薪低于本科直接就业三年后的薪资，则说明读研在短期收入层面可能没有形成优势。但这并不意味着读研长期无价值，因为部分专业的学历收益可能在更长周期内体现。

### 9.4 读研机会成本

读研机会成本是本项目解释个体体感的重要指标。

计算方式为：

读研机会成本 = 本科直接工作三年累计收入 + 读研期间净成本 - 读研带来的薪资增量折现收益

在简化展示中，也可以采用以下表达：

读研机会成本 = 本科直接工作三年累计收入 + 三年工作经验溢价 - 读研期间收入或补贴 - 硕士毕业后的薪资增量

该指标不应被视为精确财务模型，而应被理解为职业路径比较工具。它的价值在于说明：读研不是零成本选择。如果研究生阶段没有带来足够的职业资本增量，个体在毕业时就可能感受到明显的机会成本压力。

## 十、指标使用边界与解释规范

本项目中的指标主要用于结构性解释，而不是用于对个体做绝对判断。硕士岗位吸纳比低，并不意味着硕士学历没有价值；它只说明在特定岗位样本中，明确要求硕士学历的岗位占比有限。硕士学历错配指数为正，也不意味着硕士毕业生整体过剩；它只说明供给端硕士占比与需求端硕士岗位占比之间存在差距。

同样，机会成本模拟也不能直接推导出“读研不值得”。读研是否值得，取决于具体学校、专业、城市、行业、个人能力和职业规划。对于某些专业而言，研究生学历可能是进入行业的必要条件；对于另一些岗位而言，三年工作经验可能比学历提升更有市场价值。因此，本项目的结论必须始终回到结构条件，而不是对学历价值作简单化判断。

在视频和报告表达中，应尽量避免使用“研究生不值钱了”“读研没用”“硕士过剩”这类绝对化表述。更准确的表达应当是：在高等教育扩张和岗位结构分化的背景下，硕士学历的就业价值正在从单一学历信号转向复合职业资本。学历仍然重要，但它必须与专业壁垒、实习经历、项目能力、行业需求和城市选择共同发挥作用。

## 十一、核心指标汇总

本项目第一阶段应优先完成以下核心指标：

1. 高校毕业生规模，用于刻画总体供给压力。
2. 研究生毕业生规模，用于刻画高学历供给压力。
3. 硕士毕业生占比，用于观察高学历供给结构变化。
4. 应届岗位数，用于刻画招聘市场对应届毕业生的岗位池。
5. 硕士硬性吸纳比，用于衡量明确要求硕士学历的岗位占比。
6. 硕士偏好吸纳比，用于衡量企业对硕士学历的软性偏好。
7. 行业硕士吸纳比，用于比较不同行业对硕士学历的需求差异。
8. 城市硕士吸纳比，用于比较不同城市对硕士学历的吸纳能力。
9. 行业—城市硕士吸纳比，用于识别高学历岗位的空间与产业集中情况。
10. 硕士学历错配指数，用于比较硕士供给占比与硕士岗位需求占比之间的偏离。
11. 读研机会成本，用于解释本科直接就业与读研后就业之间的路径差异。

这些指标共同构成本项目的基础分析框架。后续数据清洗、图表制作和报告撰写都应围绕这些指标展开，避免在没有明确研究问题的情况下无限扩展数据维度。
