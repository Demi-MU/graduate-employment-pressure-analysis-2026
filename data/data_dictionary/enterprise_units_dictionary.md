# 企业法人单位数分省面板数据字典

对应文件：

```text
data/processed/enterprise_units_by_province_panel.csv
```

## 数据用途

该表用于刻画省级地区企业主体存量变化，作为地区就业承载能力和岗位机会分布的背景变量。它不能直接等同于招聘岗位数，也不能代表应届岗位、硕士岗位或高质量岗位数量。

## 字段说明

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| year | 年份 | 原表包含2016、2017、2019、2020、2021、2022、2024年 |
| province | 省级地区 | 原始地区名称，保留直辖市、省和自治区名称 |
| enterprise_legal_person_units | 企业法人单位数 | 单位为个 |
| unit | 单位 | 固定为“个” |
| data_scope | 数据口径 | 固定为“分省年度数据” |
| source | 数据来源 | 国家统计局及原始工作簿名 |
| source_file | 原始文件路径 | 指向 `data/raw/official_statistics/` 下的原始表 |
| note | 口径说明 | 记录缺失年份和指标使用边界 |

## 使用边界

该指标适合回答“哪些地区企业主体基础更强、变化更快”，但不能单独回答“哪些地区岗位更多”或“哪些地区硕士岗位更多”。在本项目中，它应与岗位样本数据、城市岗位密度和硕士岗位吸纳比结合使用。
